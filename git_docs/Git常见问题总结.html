# Git常见问题总结

# 一、Git 提示fatal: remote origin already exists 错误解决办法

1、先删除远程 Git 仓库

$ git remote rm origin
1

2、再添加远程 Git 仓库

$ git remote add origin git@github.com:FBing/java-code-generator
1

3、如果执行 git remote rm origin 报错的话，我们可以手动修改gitconfig文件的内容

$ vi .git/config
1
